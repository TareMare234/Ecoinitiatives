# Ecoinitiatives – Plattform für lokale & globale Nachhaltigkeitsprojekte

EcoInitiatives ist eine Single Page Application, die nachhaltige Projekte und Initiativen (Aufforstung, Clean-Up-Aktionen, Urban Gardening, Plastikreduktion etc.) aus einer API lädt und übersichtlich darstellt. Nutzer können Projekte filtern, Detailseiten mit Karte, Impact-Zahlen und Fotos ansehen. Im Extended-Modus können neue Initiativen erstellt, bestehende bearbeitet oder gelöscht werden. Die App motiviert zum Mitmachen und zeigt echten Umwelt-Impact.
